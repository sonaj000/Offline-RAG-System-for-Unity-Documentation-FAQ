    Paint tiles with the Paint tool

    Use the Paint tool to select tiles and then paint them onto the tilemapA GameObject that allows you to quickly create 2D levels using tiles and a grid overlay. More info
    See in Glossary    .

    1. Select the Paint tool from the toolbar, or press its shortcut key B.

    2. Select a tile from the Tile Palette window and paint it onto the tilemap in the SceneA Scene contains the environments and menus of your game. Think of each unique Scene file as a unique level. In each Scene, you place your environments, obstacles, and decorations, essentially designing and building your game in pieces. More info
      See in Glossary       view with the Paint tool. To paint with more than one tile, hold and drag the Paint tool over multiple tiles to select them all, then paint with them.

    3. To select tiles that are on the tilemap instead of the Tile Palette window, hold Ctrl to temporarily switch to the Pick tool and pick tiles on the tilemap. After selecting the tile(s), release Ctrl to return to the Paint tool and paint with the selected tiles.

    4. To erase tiles, hold Shift to temporarily switch to the Eraser tool and select tiles on the tilemap that you want to remove. Release Shift to return to the Paint tool**.
