    Using the Select tool

    Use the Select tool to select tiles on the tilemapA GameObject that allows you to quickly create 2D levels using tiles and a grid overlay. More info
    See in Glossary    , and use that selection with other Tile Palette editor tools such as the Move tool to move selected tiles.

    When you select tiles with the Select tool, more properties and options become available for the selected cells in the Grid Selection Properties and Modify Tilemap sections. Refer to their respective documentation to learn how you can adjust their properties and options.

    Topic Description
    Select tiles with the Select tool Use the Select tool to select tiles and cells.
    Grid Selection Properties reference Refer to the Grid Selection InspectorA Unity window that displays information about the currently selected GameObject, asset or project settings, allowing you to inspect and edit the values. More info

    Modify Tilemap reference Refer to the Modify Tilemap section’s gizmosA graphic overlay associated with a GameObject in a Scene, and displayed in the Scene View. Built-in scene tools such as the move tool are Gizmos, and you can create custom Gizmos using textures or scripting. Some Gizmos are only drawn when the GameObject is selected, while other Gizmos are drawn by the Editor regardless of which GameObjects are selected. More info
